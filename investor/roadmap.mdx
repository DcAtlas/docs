**Ten-Year North Star**

_By 2034 AIVA becomes the “silent OS” of the internet: every screen, wallet, or device instantly reshapes itself to the user’s intent, while agents swap value and data in the background. Interfaces disappear; execution is ambient._

| Quarter     | Milestone                                                                                                                                                                                                        | Why It Matters                                                             |
| ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| **Q4 2024** | <ul><li>R\&D inception → NTO + DRL PoC.</li><li>First internal live-UI demo.</li></ul>                                                                                                                           | Proves the science & locks security assumptions **before** handling value. |
| **Q1 2025** | <ul><li>**Closed Alpha** (≤ 50 wallets).</li><li>MCP Registry *v0* (manual path onboarding).</li><li>UI-credit burn live on testnet.</li></ul>                                                                   | Real usage telemetry; tune token-burn coefficients.                        |
| **Q2 2025** | <ul><li>Path SDK *v1* (TypeScript).</li><li>Tiny-Path bounties (DEX stats, gas feeds, NFT meta).</li><li>Wait-list > 5k.</li></ul>                                                                               | Bootstraps the dev flywheel before liquidity.                              |
| **Q3 2025** | <ul><li>**TGE** — AIVA token on mainnet & DEX.</li><li>Staking contract + governance *alpha*</li><li>First parameter votes (fee tweak, path approval).</li></ul>                                                 | Aligns incentives: burn ∕ stake ∕ govern now live.                         |
| **Q4 2025** | <ul><li>**Public Beta** (open sign-ups).</li><li>Path Marketplace *v1* with auto-royalties.</li><li>Mobile SDK (iOS / Android voice kit).</li></ul>                                                              | Two-sided market goes public; mobile expands TAM.                          |
| **H1 2026** | <ul><li>SaaS & CeFi connectors (G-Suite, Slack, CEX APIs).</li><li>L2 “UI-settlement” roll-up testnet (sub-cent render gas).</li></ul>                                                                           | Connects Web2 cash-flows & slashes per-render cost.                        |
| **H2 2026** | <ul><li>**DAO Phase I** — treasury & grants on-chain.</li><li>Cross-Domain Agent Mesh (agents delegate tasks P2P).</li><li>Path Marketplace *v2* (auction-based fee discovery).</li></ul>                        | Decentralises spend & lets high-value paths compete.                       |
| **H1 2027** | <ul><li>Verifiable ML network (decentralised inference pilot).</li><li>Zero-Click browser overlay (any site → AIVA UI)</li><li>Regulated fiat ramps (optional KYC modules).</li></ul>                            | Removes single-provider AI & bridges Web2 friction gaps.                   |
| **H2 2027** | <ul><li>**DAO Phase II** — full protocol upgrade control.</li><li>Multi-chain deployments (Solana, OP, Base).</li><li>Treasury-funded vertical packs (de-facto interfaces for DeFi, commerce, gaming).</li></ul> | Community steers roadmap; AIVA everywhere value moves.                     |
