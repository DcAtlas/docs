## Why It Matters

- **Visual Agility** – Users flip between table, chart, wizard, or game without reloading a single page.
- **Zero-Code Execution** – Every UI element is already wired to wallets, paths, and on-chain safety checks.
- **Context Carry-Over** – AIVA remembers parameters across views (e.g., token list → swap) so users never re-enter data.
